一、在创建线程时传递参数

代码演示

  在初始化子线程时，第一个参数为 可调用对象，之后依次填入函数需要的参数。

class A {
public:
  void operator()(int x) {
      fun(x);
  }  
};
void fun(int x) {
    cout << "线程开始执行\n";
 
    cout << x << endl;

    cout << "线程执行完毕\n";
}
int main(void)
{
    int x = 5;
    //函数
    thread myt1(fun, x);
    myt1.join();

    //类
    A a(x);
    thread myt2(a, x);
    myt2.join();

    //lambda
    auto mylambda = [] (int x) {
        cout << "线程开始执行\n";
        cout << x << endl;
        cout << "线程执行完毕\n";
    };
    thread myt3(mylambda, i);
    myt3.join();

    cout << "main执行完毕\n";
    return 0;
}

传递的参数为引用或指针可能引发的错误

void fun(const int &i, char *string) {
    cout << "线程开始执行\n";
 
    cout << i << "\n" << string << "\n";

    cout << "线程执行完毕\n";
}
int main() {
    int i = 4;
    int& v1 = i;
    char buf[] = "testtest";

    thread myt1(fun, v1, buf);
    myt1.detach();
    return 0;
}

  经过调试，发现在子线程内第一个引用参数 $i$ 和主线程的 $v1$ 地址不同，因此调用detach不会发生错误，但不推荐用引用，直接传值即可。而第二个参数是个指针，在子线程和主线程的地址相同，这时如果再调用detach()程序就会发生错误了。为了能将该字符串传入子线程，我们用下面这种方法：

// 在这里的形参得加上引用，否则会在子线程多复制一次（多调用一次拷贝构造函数）  
// 同时在函数内对buf的修改不会引起主线程buf的修改
void fun(const int &i, const string &buf) {
    cout << "线程开始执行\n";
 
    cout << i << "\n" << buf << "\n";

    cout << "线程执行完毕\n";
}
int main(void)
{
    int i = 4;
    int& v1 = i;
    char buf[] = "testtest";

    thread myt(fun, v1, (string)buf);

    myt.join();
    cout << "main执行完毕\n";
    return 0;
}

  需要注意的是，在创建线程时的强转类型一定要加上，这样才会在创建线程时用主线程的buf创建好string类型的临时变量。否则，主线程的buf很有可能在主线程结束回收资源之后，才被子线程的buf拿来复制，而很明显此时，主线程的buf已经不存在了。

二、线程id

概念

  线程id是一个数字，一个线程对应一个id。

输出当前所在线程id

    cout << std::this_thread::get_id() << endl;

  通过输出当前线程id，我们能更容易进行代码调试。

三、std::ref函数

  在之前我们说过，即使在函数头部给形参加了&，函数内的修改还是不会引起外部修改。为了达到能同时修改的目的，我们可以使用std::ref函数。这时候在主线程中创建子线程时也就不会调用拷贝构造函数，因此使用std::ref的前提是子线程调用join()。

class A {
public:
    int i;
    A(int _i) : i(_i) {}
};

void fun(A &a) {
    cout << "线程开始执行\n";

    a.i = 13; // 同时主线程的a也被修改了
    cout << a.i << "\n";

    cout << "线程执行完毕\n";
}
int main(void)
{
    A a(12);
    thread myt(fun, std::ref(a));

    myt.join();
    cout << "main执行完毕\n";
    return 0;
}
